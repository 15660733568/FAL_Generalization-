## 🔧 1. Environment Setup

### Requirements
* Python >= 3.8

### Installation

```bash
# Create and activate a virtual environment (using Python 3.8)
conda create -n minintp python=3.8
conda activate minintp

# Install PyTorch and Torchvision
# (Note: This installs the default version. For specific CUDA version, please refer to pytorch.org)
pip install torch torchvision

# Install scientific computing, data analysis, and machine learning libraries
pip install numpy pandas scikit-learn

# Install visualization tools
pip install matplotlib seaborn

# Install utility and computer vision packages
pip install tqdm opencv-python
